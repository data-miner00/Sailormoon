# Sailormoon

Sailormoon is a Discord bot made with Discord.js API that serves to remind friends upon Discord Nitro expiry.

## Folder structure

Initial folder structure

```
Sailormoon
┬
└-- src
│   └-- commands
│   │    └-- index.ts
│   └-- events
│   │    └-- index.ts
│   │    └-- message.ts
│   │    └-- ready.ts
│   └-- app.ts
│   └-- config.ts
└-- .env
└-- .env.example
└-- .gitignore
└-- package-lock.json
└-- package.json
└-- Procfile
└-- README.md
└-- tsconfig.json
```

## Scripts

- Start a local development server

```
npm run dev
```

- Building for deploymeny

```
npm run build
```
